How it works

  1. The app detects the largest visible face in each photo.
  2. It computes a crop that keeps that head centered while preserving your requested aspect ratio.
  3. It resizes the crop to the exact output dimensions and stores the DPI metadata.
  4. You receive a zip file with the processed images and a plain-text report.

If a face is not detected, the tool falls back to a centered crop so the batch still completes.
